# Zippy's Amazing AI Adventure 🤖✨

**A Kid's Guide to Artificial Intelligence**

This repo contains all creative assets, social media content, and scripts for the *Zippy's Amazing AI Adventure* book series — a children's book series for ages 6-12.

---

## 📚 About the Book

*Zippy's Amazing AI Adventure* follows six friends — the AI Explorer Squad — as they learn all about Artificial Intelligence in a fun, accessible way.
